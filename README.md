# GLM-OCR VizWiz Fine-Tuning

Fine-tuning GLM-OCR on the VizWiz dataset using LoRA and LLaMA-Factory.

## Results

### Base GLM-OCR

- Exact Match: 0.01
- CER: 4.96
- WER: 7.04

### LoRA Fine-Tuned (100 Samples)

- Exact Match: 0.33
- CER: 0.64
- WER: 0.84

## Training

- Model: GLM-OCR
- Method: LoRA
- Framework: LLaMA-Factory
- Dataset: VizWiz
